# testfastapi

### For install and run
1. Install
   - ```install_win.bat``` (for Windows)
   - ```bash install_win.sh``` (for Linux)
2. Run
   - ```run.bat``` (for Windows)
   - ```bash run.sh``` (for Linux)
3. Create testdata
   - open new terminal (cmd)
   - activate env (if not activated)
   - ```python testdata.py```
